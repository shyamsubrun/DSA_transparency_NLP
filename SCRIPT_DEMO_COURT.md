# Script Court pour Démo (Version Rapide - 5 minutes)

## Introduction (20 sec)
"Bonjour, je vais vous présenter le dashboard DSA que j'ai développé en frontend. C'est une interface pour visualiser les données de modération de contenu des plateformes numériques."

---

## Vue d'ensemble (30 sec)
"Le dashboard affiche les décisions de modération prises par Meta, TikTok, X, YouTube, etc. J'ai utilisé React, TypeScript, et ECharts pour créer une interface moderne et interactive."

---

## Fonctionnalités principales (2 min)

### KPIs
"En haut, les indicateurs clés : nombre total d'actions, plateformes surveillées, délai moyen, taux d'automatisation. Ils se mettent à jour avec les filtres."

### Filtres
"Le panneau de filtres permet de filtrer par dates, plateformes, catégories, types de décision, pays, etc. Tout fonctionne en temps réel."

### Graphiques
"Plusieurs visualisations interactives : évolution temporelle, répartition par plateforme, géographie, catégories. Tous réactifs aux filtres."

### Responsive
"Le dashboard s'adapte aux mobiles, tablettes et desktop."

---

## Démo interactive (2 min)
*[Montrer en direct :]*
1. Vue générale des KPIs
2. Application de filtres (ex: Meta + dernier trimestre)
3. Exploration des graphiques interactifs
4. Test de la responsivité

---

## Points forts (30 sec)
"Architecture modulaire, performance optimisée, interface intuitive, code TypeScript type-safe, intégration fluide avec le backend."

---

## Conclusion (20 sec)
"J'ai créé un dashboard moderne et performant pour visualiser les données DSA. Prêt pour la production. Des questions ?"
